# Гистограмма - программа для анализа данных

Программа для построения гистограмм с графическим интерфейсом, статистическим анализом и визуализацией данных.

##  Возможности

-  Построение гистограмм с настраиваемым количеством интервалов
-  Статистический анализ данных (среднее, медиана, дисперсия и др.)
-  Загрузка данных из файлов, ручной ввод и генерация случайных чисел
-  Темная и светлая темы интерфейса
-  Сохранение гистограмм в виде изображений
-  Экспорт результатов в текстовые файлы

##  Установка и запуск

### Требования
- Python 3.7+
- Установленные зависимости из `requirements.txt`

### Установка зависимостей
```bash
pip install -r requirements.txt
